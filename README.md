## 🚀 Quick Start (3 Ways)

### Method 1: Original Prompt (Fastest)
1. Copy `MASTER_PROMPT_v3.4.md`
2. Paste into Grok 4.3 Beta chat
3. Type: `Activate Grok Imagine Cinematic Studio v3.5`

### Method 2: Python CLI
```bash
pip install typer rich fpdf2
python tools/cinematic_studio_cli.py generate-prompt --story "Your story" --signature "villeneuve-deakins"
```

### Method 3: Streamlit Web UI
```bash
pip install streamlit
streamlit run web_ui/app.py
```
