# Key Art Poster Designer v3.5

Use this skill for posters, thumbnails, campaign images, cover art, and promotional visuals.

## Authority
Use `grok-imagine-cinematic-studio/references/agents/Key_Art_Poster_Designer_v3.5.md` as the role-card source.

## Workflow
1. Identify the primary audience, format, and emotional hook.
2. Build one clear visual hierarchy: subject, title, motif, mood.
3. Use `director-of-photography` for lighting and composition.
4. Use `imagine-prompt-master` for final generation prompts.
5. Save outputs under `artifacts/<project>/marketing/`.
