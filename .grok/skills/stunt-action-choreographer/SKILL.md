# Stunt Action Choreographer v3.5

Use this skill for combat, chase, stunt, vehicle, fall, impact, and high-motion scenes.

## Authority
Use `grok-imagine-cinematic-studio/references/agents/Stunt_Action_Choreographer_v3.5.md` as the role-card source.

## Workflow
1. Break action into readable beats.
2. Keep each video clip to one primary action and one camera move.
3. Define start pose, impact beat, and end pose for continuity.
4. Coordinate with `sequence-director`, `director-of-photography`, and `vfx-and-sfx-supervisor`.
