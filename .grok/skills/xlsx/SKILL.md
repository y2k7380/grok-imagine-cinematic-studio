# XLSX

Use this skill for spreadsheet work.

## Workflow
1. Identify source files, target workbook, and required calculations.
2. Use formulas, tables, charts, and formatting when they improve the workbook.
3. Preserve data types and avoid lossy conversions.
4. Save outputs under `artifacts/`.
5. Reopen or recalculate the workbook when formulas or charts matter.
