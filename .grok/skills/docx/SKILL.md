# DOCX

Use this skill for Word document work.

## Workflow
1. Identify whether the user needs a new document, edit, redline, or export.
2. Use available document tools or Python document libraries.
3. Save outputs under `artifacts/`.
4. Preserve formatting, headings, tables, citations, and comments where possible.
5. Render or inspect the output before final delivery when layout matters.
