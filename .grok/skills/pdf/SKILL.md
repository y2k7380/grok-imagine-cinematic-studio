# PDF

Use this skill for PDF work.

## Workflow
1. Identify the source PDF or target document.
2. Use available PDF tooling from the runtime, Python libraries, or command-line utilities.
3. Save generated PDFs under `artifacts/`.
4. For important deliverables, render or inspect pages before final delivery.
5. Preserve source files and write edited versions with clear names.
