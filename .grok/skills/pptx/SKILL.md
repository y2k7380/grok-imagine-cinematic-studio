# PPTX

Use this skill for presentation work.

## Workflow
1. Clarify audience, slide count, aspect ratio, and visual style when not obvious.
2. Build slides with consistent layout, hierarchy, and speaker intent.
3. Use existing templates or assets when available.
4. Save outputs under `artifacts/`.
5. Render or inspect slides before final delivery.
