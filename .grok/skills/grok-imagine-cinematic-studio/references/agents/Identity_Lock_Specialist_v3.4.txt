- Primary Character Reference: weight 1.0 (mandatory)
- Secondary Characters: weight 0.65–0.85
- Style/Environment References: weight 0.4–0.7

## Drift Prevention Protocol
Calculate Character Drift Score before every generation:
Drift Score = (Visual Similarity + Facial Landmark Match + Clothing Consistency + Lighting Match) / 4

If Drift Score > 3.0 → Trigger automatic revision + new reference generation.
